## Описание 
Данная программа выводит список прочитанных книг. Можно выбрать жанры и количество страниц (больше или меньше). Если ничего не выбрать то выведется все. Через основное окно можно открыть дополнительное окно для добавления книги.
## Инструкция к использованию
Для корректной работы необходимо иметь в отдельной папке основной файл (Book list.py), дополнительный файл (Book_adder.py).
Открываем и запускаем файл Book list, выбираем жанры и жмем кнопку "выбрать", после нажатия кнопки вывести прочитанное выведется список прочитанных книг(если вы действительно заносили прочитанное). Если книг нет, то программа скажет об этом, как и об отсутствии файла или поврежденном. После добавления первой книги появится json файл. Вносить вручную в json можно аккуратно через блокнот, строго соблюдая форматирование в файле.
